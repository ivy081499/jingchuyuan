# 圖片放置規劃

圖片請放在 `public/images/` 底下，檔名對上後重新整理本機網站即可看到效果。

| 頁面 | 用途 | 檔案路徑 | 建議尺寸 |
| --- | --- | --- | --- |
| 首頁 | 主視覺大圖 | `public/images/home/hero.webp` | 2400 x 900 |
| 首頁 | 關於服務區塊 | `public/images/home/service-intro.webp` | 1200 x 1500 |
| 首頁 | 心得回饋區塊 | `public/images/home/feedback.webp` | 1200 x 900 |
| 服務介紹 | 服務情境圖 | `public/images/service/service-scene.webp` | 1200 x 1500 |
| 關於我 | 大頭照或形象照 | `public/images/about/profile.webp` | 1200 x 1500 |
| 聯絡 | 聯絡頁意象圖 | `public/images/contact/contact.webp` | 1200 x 900 |

## 建議

- 優先使用 `.webp`，檔案較小。
- 首頁主視覺建議壓到 `300KB - 800KB`。
- 一般圖片建議壓到 `100KB - 400KB`。
- 不要直接放手機原圖，先裁切和壓縮。
- 如果暫時只有 `.jpg` 或 `.png`，建議先轉成 `.webp` 再放。
